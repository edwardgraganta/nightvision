The "Five-Year Dark" Principle
I have maintained a minimal social media presence for over five years. This "Digital Detachment" has allowed me to:

Reduce Attack Surface: Minimizing public personal data that could be used for social engineering.

Focus on Deep Work: Prioritizing technical skill-building over the "scroll culture" of modern platforms.

Data Sovereignty: Maintaining control over my personal information and where it is stored.
